日本語

収集する情報

本アプリ(以下「本アプリ」)は、サーバーを持たず、アカウント登録も不要です。すべての記録(参拝記録、写真、メモ、設定)は、お使いの端末内にのみ保存されます。開発者および第三者がこれらのデータにアクセスすることはありません。

位置情報

近くの神社・お寺・お城を表示するために、位置情報の利用許可をお願いする場合があります。位置情報は端末内での距離計算にのみ使用され、外部に送信・保存されることはありません。許可しなくてもアプリの主要機能は利用できます。

写真

参拝の思い出を記録するために、カメラおよびフォトライブラリへのアクセス許可をお願いする場合があります。選択した写真はアプリ内の書類フォルダにコピーされ、端末内にのみ保存されます。

通知

参拝リマインダー(月参り・お礼参り)は、端末内で完結するローカル通知です。設定画面からいつでもオフにできます。

地図データ

地図の表示には OpenFreeMap のタイルサーバーを利用します。地図タイルの取得時に、IP アドレス等の接続情報がタイルサーバーに送信されます(一般的な Web アクセスと同等)。本アプリ自体がユーザーを識別する情報を送信することはありません。

内蔵の社寺データは © OpenStreetMap contributors(ODbL ライセンス)に基づきます。

解析・広告

アクセス解析ツール・広告 SDK は一切使用していません。

データのバックアップと削除

設定画面からすべてのデータを zip ファイルとして書き出せます。アプリを削除すると、端末内のすべてのデータが削除されます。

お問い合わせ

ご質問は App Store のサポート連絡先までお寄せください。

繁體中文

我們收集什麼資訊

本應用程式沒有伺服器、無需註冊帳號。所有記錄(參拜記錄、照片、筆記、設定)只儲存在你的裝置上。開發者與任何第三方都無法存取這些資料。

位置資訊

為了顯示附近的神社、寺院與城堡,應用程式可能請求位置權限。位置資訊只用於裝置上的距離計算,不會傳送或儲存到任何外部伺服器。不授權也能使用主要功能。

照片

為了記錄參拜回憶,應用程式可能請求相機與照片圖庫權限。所選照片會複製到應用程式的文件資料夾,僅保存在裝置內。

通知

參拜提醒(月參り/還願提醒)為純本地通知,可隨時在設定頁關閉。

地圖資料

地圖顯示使用 OpenFreeMap 的圖磚伺服器。取得圖磚時,連線資訊(如 IP 位址)會傳送至圖磚伺服器(與一般網頁瀏覽相同)。本應用程式不會傳送任何可識別使用者的資訊。

內建社寺資料基於 © OpenStreetMap contributors(ODbL 授權)。

分析與廣告

本應用程式完全不使用分析工具或廣告 SDK。

資料備份與刪除

可在設定頁將全部資料匯出為 zip 檔。刪除應用程式即刪除裝置內所有資料。

聯絡方式

如有疑問,請透過 App Store 的支援聯絡方式與我們聯繫。
